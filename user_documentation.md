## Авторизация
Для использования данного музыкального сервиса требуется **авторизироваться**. При запуске приложения пользователя встретит окно с двумя полями для логина и пароля, а также кнопкой, нажимая на которую пользователь авторизуется при условии верности введённых данных. После этого пользователь попадает на **[Главное меню приложения](#Главное-меню-приложения)**.

## Главный экран приложения:
В главном экране пользователь увидит кнопку **выбор плейлиста** в левом верхнем углу экрана, **настройки профиля** в правом верхнем углу экрана. По центру верхней части экрана будет располагаться **Поле для поиска треков**. Прямо под ним будут располагаться треки плейлиста.

- ### Выбор плейлиста
**⚠ Данная функция в разработке ⚠**

Выбор плейлиста - это кнопка, расположенная в правом верхнем углу экрана, которая позволяет выбирать плейлист, треки которого будут отображаться на главном экране.

- ### Профиль
**⚠ Данная функция в разработке ⚠**

У пользователя есть возможность настраивать свой профиль при нажатии по аватарке в правом верхнем углу экрана. Можно изменить ник, отображаемый для других пользователей, аватарку, пароль и логин.

- ### Поиск
**⚠ Данная функция в разработке ⚠**

Функция поиска представляет из себя пустое поле, свободное для заполнения символами, находящееся в верхней части экрана между иконкой-аватаркой и кнопкой для выбора плейлистов, а также кнопка с символом лупы, нажимая на которую происходит поиск по заданной в текстовом поле строке. 

- ### Добавление треков
Добавление треков визуально реализовано как символ "плюс", находящийся под текстовым полем для поиска с левой стороны экрана. При нажатии открывает окно с настройкой добавления трека. В настройке добавление трека реализовано указание альтернативного имени и указание плейлиста, в котором будет находиться добавляемый трек, а также применяемые функции редактирования.
